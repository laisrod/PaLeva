# 🍽️ PaLeva - Sistema de Gestão para Restaurantes

Acabei de finalizar o desenvolvimento do **PaLeva**, uma aplicação full-stack completa para gestão de restaurantes que conecta proprietários e clientes em uma plataforma moderna e intuitiva.

## 🎯 O Projeto

O PaLeva é um sistema completo que permite aos restaurantes gerenciarem seus cardápios, pedidos, estatísticas e interações com clientes, enquanto oferece uma experiência fluida para os consumidores navegarem, fazerem pedidos e acompanharem suas compras em tempo real.

## 🚀 Principais Funcionalidades

### Para Proprietários:
✅ **Dashboard completo** com estatísticas e métricas de negócio
✅ **Gestão de cardápios** - criação e edição de pratos, bebidas e porções
✅ **Controle de pedidos** em tempo real com atualizações de status
✅ **Gestão de estabelecimento** - configuração de horários e informações
✅ **Sistema de tags** para organização de itens
✅ **Avaliações e reviews** dos clientes

### Para Clientes:
✅ **Descoberta de restaurantes** com interface intuitiva
✅ **Navegação de cardápios** com filtros e categorias
✅ **Carrinho de compras** e realização de pedidos
✅ **Histórico de pedidos** e acompanhamento em tempo real
✅ **Notificações em tempo real** sobre status dos pedidos

## 🛠️ Stack Tecnológica

### Backend:
- **Ruby on Rails 7.2** - API RESTful robusta
- **Action Cable** - WebSockets para atualizações em tempo real
- **Devise** - Sistema de autenticação
- **PostgreSQL** - Banco de dados relacional

### Frontend:
- **React 18** com **TypeScript** - Interface moderna e type-safe
- **Vite** - Build tool de alta performance
- **React Router** - Navegação SPA
- **WebSocket** - Comunicação em tempo real

### DevOps:
- **Docker & Docker Compose** - Containerização
- **Render** - Deploy do backend
- **Vercel** - Deploy do frontend

## 💡 Desafios Técnicos Superados

🔧 **Integração WebSocket em produção** - Implementação de notificações em tempo real com tratamento robusto de erros e reconexão automática

🔧 **Arquitetura modular** - Separação clara entre área do cliente e proprietário, facilitando manutenção e escalabilidade

🔧 **Gestão de estado complexa** - Implementação de hooks customizados e contextos para gerenciar carrinho, autenticação e notificações

🔧 **API RESTful bem estruturada** - Serializers customizados, tratamento de erros padronizado e autenticação via token

## 📊 Destaques do Projeto

- ✅ Autenticação segura com JWT
